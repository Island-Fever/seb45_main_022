## ✨ 커밋 컨벤션

```
✨ feat: 새로운 기능 추가
🐛 fix: 오류 또는 버그 수정
📝 docs: `readme.md`와 같은 문서 관련 변경사항
🎨 style: 코드 스타일 변화
💄 design: CSS 등 사용자 UI 디자인 변화
♻️ refactor: 기능과 관련없는 코드의 리팩토링
🥽 test: 테스트를 추가하거나 수정했을 때
⚙️ chore: build와 관련된 부분, 패키지 매니저 설정 등
```
